# نظام إدارة الكورسات - لوحة الإدارة

## نظرة عامة

تم تطوير نظام شامل لإدارة الكورسات في لوحة الإدارة يتيح للمديرين عرض تفاصيل الكورسات والمعلمين والجدولة والطلاب المسجلين، بالإضافة إلى إمكانية التعديل والإدارة الكاملة.

## الميزات الرئيسية

### 1. عرض تفاصيل الكورس (Show.vue)
- **المعلومات الأساسية**: العنوان، الحالة، المستوى، السعر، المدة، الحد الأقصى للطلاب
- **تفاصيل المعلم**: اسم المعلم، البريد الإلكتروني
- **الجدولة**: عرض جميع مواعيد الكورس مع الأيام والأوقات
- **المتطلبات والنتائج التعليمية**: قوائم قابلة للتعديل
- **إحصائيات الكورس**: عدد الطلاب المسجلين، الأماكن المتاحة
- **قائمة الطلاب المسجلين**: عرض تفاصيل كل طالب مع التقدم والحالة

### 2. تعديل الكورس (Edit.vue)
- **نموذج تعديل شامل**: جميع الحقول قابلة للتعديل
- **إدارة الجدولة**: إضافة/حذف/تعديل مواعيد الكورس
- **إدارة المتطلبات**: إضافة/حذف المتطلبات
- **إدارة النتائج التعليمية**: إضافة/حذف النتائج التعليمية
- **التحقق من صحة البيانات**: تحقق شامل من جميع الحقول

### 3. إدارة الطلاب المسجلين
- **عرض التسجيلات**: جدول شامل لجميع الطلاب المسجلين
- **معلومات الطالب**: الاسم، البريد الإلكتروني، تاريخ التسجيل
- **حالة التسجيل**: مسجل، مكتمل، منسحب
- **متابعة التقدم**: شريط تقدم بصري مع النسبة المئوية
- **إجراءات سريعة**: تعديل التسجيل، إضافة طالب جديد

## البنية التقنية

### الملفات الرئيسية
```
resources/js/Pages/Admin/Courses/
├── Show.vue          # صفحة عرض تفاصيل الكورس
├── Edit.vue          # صفحة تعديل الكورس
├── Create.vue        # صفحة إنشاء كورس جديد
└── Index.vue         # صفحة قائمة الكورسات
```

### النماذج (Models)
- **Course**: نموذج الكورس مع العلاقات
- **CourseSchedule**: نموذج جدولة الكورس
- **CourseEnrollment**: نموذج تسجيل الطلاب
- **User**: نموذج المستخدم (معلم/طالب)

### المتحكمات (Controllers)
- **CourseController**: إدارة الكورسات (CRUD)
- **EnrollmentController**: إدارة التسجيلات

## الطرق (Routes)

### إدارة الكورسات
```php
Route::resource('courses', CourseController::class);
// GET    /admin/courses              - عرض قائمة الكورسات
// GET    /admin/courses/create       - إنشاء كورس جديد
// POST   /admin/courses              - حفظ الكورس
// GET    /admin/courses/{id}         - عرض تفاصيل الكورس
// GET    /admin/courses/{id}/edit    - تعديل الكورس
// PUT    /admin/courses/{id}         - تحديث الكورس
// DELETE /admin/courses/{id}         - حذف الكورس
```

### إدارة التسجيلات
```php
Route::resource('enrollments', EnrollmentController::class);
// GET    /admin/enrollments              - عرض قائمة التسجيلات
// GET    /admin/enrollments/create       - إنشاء تسجيل جديد
// POST   /admin/enrollments              - حفظ التسجيل
// GET    /admin/enrollments/{id}/edit    - تعديل التسجيل
// PUT    /admin/enrollments/{id}         - تحديث التسجيل
// DELETE /admin/enrollments/{id}         - حذف التسجيل
```

### 2. التصميم المتجاوب
- **Grid System**: تخطيط مرن يتكيف مع جميع الشاشات
- **Dark Mode**: دعم الوضع المظلم
- **Tailwind CSS**: تصميم حديث وجميل

### 3. التفاعل مع المستخدم
- **رسائل الخطأ**: عرض أخطاء التحقق من صحة البيانات
- **رسائل النجاح**: تأكيد العمليات الناجحة
- **تأكيد الحذف**: تأكيد قبل حذف الكورس

## كيفية الاستخدام

### 1. عرض تفاصيل الكورس
1. انتقل إلى لوحة الإدارة
2. اختر "إدارة الكورسات"
3. اضغط على "عرض" بجانب الكورس المطلوب
4. ستظهر صفحة شاملة بجميع التفاصيل

### 2. تعديل الكورس
1. من صفحة تفاصيل الكورس، اضغط "تعديل الكورس"
2. عدل الحقول المطلوبة
3. أضف/احذف المواعيد حسب الحاجة
4. اضغط "تحديث الكورس"

### 3. إدارة الطلاب
1. من صفحة تفاصيل الكورس، ستجد قائمة الطلاب المسجلين
2. اضغط "إضافة طالب" لإضافة طالب جديد
3. اضغط "تعديل" بجانب أي طالب لتعديل تسجيله

## الأمان والصلاحيات

### التحقق من الصلاحيات
- **Middleware**: `auth` و `role.redirect`
- **التحقق من الدور**: فقط المديرون يمكنهم الوصول
- **CSRF Protection**: حماية من هجمات CSRF

### التحقق من صحة البيانات
- **Validation Rules**: قواعد تحقق شاملة
- **Sanitization**: تنظيف المدخلات
- **SQL Injection Protection**: حماية من حقن SQL

تم تطوير نظام إدارة كامل وشامل للكورسات يوفر:
- عرض تفصيلي لجميع معلومات الكورس
- إمكانية التعديل الكامل للكورس
- إدارة شاملة للطلاب المسجلين

هذا النظام يوفر للمديرين جميع الأدوات اللازمة لإدارة الكورسات بكفاءة وفعالية.
